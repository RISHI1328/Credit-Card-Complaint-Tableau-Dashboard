This project presents an interactive Tableau dashboard analyzing consumer credit card complaints across the United States.
The goal is to identify trends, complaint categories, state-wise patterns, company responses and how effectively complaints are resolved.

This dashboard converts raw complaint records into a clear, actionable, and visual summary that assists in operational monitoring and decision-making.
